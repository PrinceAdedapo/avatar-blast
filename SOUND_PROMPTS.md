### Recommended Formats
- **Voice lines:** MP3, 44.1kHz, mono, ~64-128kbps (keep files small)
- **SFX:** MP3, 44.1kHz, mono, 128kbps
- **Music:** MP3, 44.1kHz, stereo, 192kbps (better quality for loops)

### Platform Tips

**Google AI Studio (Gemini TTS):**
- Best for natural-sounding character voices
- Use the voice description from the table above as system instructions
- Generate each line separately for clean cuts
- Request WAV output, convert to MP3 after

**ElevenLabs:**
- Best quality TTS, use Voice Design to create each character
- Zuko: Stability 0.4, Similarity 0.7 (more variation = more emotional)
- Aang: Stability 0.6, Similarity 0.8 (consistent cheerfulness)
- Toph: Stability 0.3, Similarity 0.6 (wild, unpredictable energy)
- Set "Style" slider high for more expressive delivery

**Suno (for music & SFX):**
- Use v4 for best quality
- Add "instrumental" to ALL music prompts to avoid vocals
- For loops: generate 2-minute versions, trim to loop point
- For SFX: set duration to "shortest" and describe precisely
- Use negative prompts: "no vocals, no singing, no lyrics"

**Udio (alternative for music):**
- Tends to produce more cinematic/orchestral results
- Better for the epic fanfares and era transitions
- Use "extend" feature to make seamless loops
