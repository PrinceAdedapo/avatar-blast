## Platform Notes
Use these on: **Google ImageFX**, **Midjourney**, **Leonardo AI**, **DALL-E 3**, or **Ideogram**
- Add `--ar 1:1` on Midjourney for square tiles
- On Google ImageFX, just paste the prompt directly
- For transparent backgrounds, generate on solid color then remove bg with remove.bg or Canva
- Generate at highest resolution available, then resize down for crisp game tiles

## 📐 GENERATION TIPS

### Size & Format Guide
| Asset Type | Ideal Size | Format | Notes |
|------------|-----------|--------|-------|
| Gem tiles (normal) | 256×256px | PNG w/ transparency | Remove bg after generation |
| Gem tiles (special) | 256×256px | PNG w/ transparency | Brighter, more energy effects |
| Backgrounds | 1080×1920px | JPG or PNG | Portrait for mobile, dark tones |
| Blast effects | 512×512px | PNG w/ transparency | Center the burst, clear edges |
| UI icons | 128×128px | PNG w/ transparency | Clean, simple, readable |
| Logo | 1024×512px | PNG w/ transparency | Wide format for header |

### Prompt Tips for Consistency
1. **Always include**: "Avatar: The Last Airbender anime art style" or "anime cel-shaded art style"
2. **For tiles**: Add "game icon asset, circular frame, clean sharp lines"
3. **For backgrounds**: Add "dark moody lighting, suitable for mobile game background"
4. **For effects**: Add "dark background, game VFX asset, transparent edges"
5. **Color consistency**: Always mention the specific character's color palette
6. **No text**: Always add "no text" unless you want text in the image
7. **Resolution**: Always end with "4K resolution" or "high detail"

### Post-Processing Workflow
1. Generate images at max resolution
2. Use **remove.bg** or **Canva Background Remover** for transparent PNGs
3. Resize to game dimensions (gem tiles → 96×96 or 128×128 for retina)
4. Add consistent border/glow in CSS (not baked into image)
5. For backgrounds: darken overlay in CSS to ensure readability
6. Test all assets at actual game size before finalizing

### Style Keywords That Work Well
- "anime cel-shaded" — matches Avatar's style
- "thick clean outlines" — readable at small sizes
- "vibrant saturated colors" — pops on dark backgrounds
- "chibi-style" — cute proportions for game tiles
- "front-facing portrait bust" — consistent angle across all tiles
- "radial energy background" — consistent framing for tiles
- "no text, no watermark" — clean assets
